# 📌 Módulo: Microsoft Copilot

Este módulo aborda o ecossistema Microsoft Copilot, suas diferentes versões e aplicações para profissionais de TI, desenvolvedores e áreas de negócio, além de um exemplo prático de análise de dados com Excel.

---

# 🎯 Objetivos do Módulo

Ao final deste módulo, você será capaz de:

- Compreender o ecossistema Microsoft Copilot e suas diferentes versões
- Identificar qual Copilot é mais adequado para cada necessidade
- Utilizar o Copilot para análise de dados em planilhas Excel
- Compreender os riscos de exposição de dados sensíveis à IA

---

# 📚 Conteúdos Abordados

## 🔹 Hub de Aprendizado do Microsoft Copilot

O Microsoft Copilot Learning Hub foi desenvolvido para profissionais de TI e desenvolvedores. O hub fornece informações detalhadas sobre o Microsoft Copilot, ajudando a entender qual Copilot é mais adequado para cada necessidade. Oferece documentação técnica, referências de produtos e módulos de treinamento para otimização de fluxos de trabalho e implementação de soluções orientadas por IA.

## 🔹 Microsoft 365 Copilot

Integrado ao pacote Office (Word, Excel, PowerPoint, Outlook, Teams). Ajuda na criação de documentos, resumos de e-mails, geração de apresentações e análise de dados, atuando diretamente dentro dos aplicativos do dia a dia.

## 🔹 Copilot no Dynamics 365

Focado em aplicativos de negócio como CRM e ERP. Auxilia na automação de processos de vendas, atendimento ao cliente, finanças e operações, trazendo IA generativa para os fluxos de trabalho empresariais.

## 🔹 Copilot no Power Platform

Integrado ao Power Apps, Power Automate e Power BI. Ajuda na criação de aplicativos, automação de fluxos de trabalho e geração de insights em dashboards de forma mais acessível e rápida.

## 🔹 Microsoft Copilot Studio

Permite criar agentes de IA personalizados (copilotos customizados) para necessidades específicas da organização. Conecta-se a sistemas internos e fontes de dados próprias.

## 🔹 Microsoft Security Copilot

Focado em segurança cibernética. Ajuda equipes de SOC (Security Operations Center) a detectar ameaças, investigar incidentes e responder a ataques com mais velocidade e eficiência.

## 🔹 Copilot no Microsoft Fabric

Integrado à plataforma de dados Microsoft Fabric. Auxilia na análise de dados, geração de insights e criação de visualizações dentro do ambiente de engenharia de dados e BI.

## 🔹 Copilot no Azure

Assistente de IA para gerenciamento de nuvem. Ajuda a criar, implantar e gerenciar recursos na Azure, além de solucionar problemas e otimizar custos.

## 🔹 GitHub Copilot

Assistente de programação integrado ao VS Code e outras IDEs. Sugere código, autocompleta funções e ajuda na documentação, acelerando o desenvolvimento de software.

## 🔹 Analisando Tabelas com o Copilot

### Passo a passo prático:

1. **Criar uma tabela no Excel** com as colunas: Loja e Vendas
2. **Popular a tabela** com dados: Lojas A, B e C e valores aleatórios de vendas
3. **Salvar o arquivo** em formato compatível (.xlsx)
4. **Arrastar o arquivo** do computador para o chat do Microsoft Copilot
5. **Solicitar a análise** com um prompt como: "Realize uma análise desta planilha e gere a soma dos valores de vendas das lojas"

# ⚠️ Pontos de Atenção

### ⚠️ CUIDADO AO EXPOR BASES DE DADOS PARA INTELIGÊNCIA ARTIFICIAL ⚠️

- Dados sensíveis podem ser processados e armazenados em servidores externos
- Nunca envie dados pessoais, financeiros ou estratégicos para IAs públicas
- Utilize versões empresariais com proteção de dados comerciais
- Anonimize dados sensíveis antes de qualquer análise externa
