# 📌 Módulo: Análise de Dados com Excel e Copilot

Este módulo consolida os fundamentos de análise de dados utilizando Microsoft Excel 365, Power Query e Microsoft Copilot, abrangendo desde conceitos básicos até a criação de dashboards interativos.

O conteúdo foi estruturado para desenvolver a base necessária para atuação em áreas como:

- Análise de Dados
- Business Intelligence (BI)
- Engenharia de Dados
- Inteligência Artificial aplicada a planilhas

A abordagem prioriza entendimento prático, com foco em automação, tratamento de dados e visualização de informações.

---

# 🎯 Objetivo do Módulo

Capacitar o uso das principais ferramentas do ecossistema Microsoft para análise de dados, permitindo:

- Domínio dos conceitos fundamentais do Excel 365
- Utilização do Microsoft Copilot para automação e análise
- Aplicação de técnicas de ETL com Power Query
- Compreensão dos conceitos de dashboards, OKRs e KPIs
- Preparação para projetos práticos

---

O conteúdo está dividido em módulos progressivos, abordando desde conceitos introdutórios do Excel até dashboards:

Apresenta os conceitos fundamentais do Microsoft Excel 365:

- Formatação de dados
- Menus e menus de contexto
