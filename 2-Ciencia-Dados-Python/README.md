# 📌 Módulo: Ciência de Dados com Python

Este módulo consolida os fundamentos da linguagem Python voltados para manipulação de dados, estruturas fundamentais e construção de lógica computacional.

O conteúdo foi estruturado para desenvolver a base necessária para atuação em áreas como:

- Engenharia de Dados
- Análise de Dados
- Machine Learning

A abordagem prioriza entendimento prático, com foco em escrita de código eficiente, legível e escalável.

---

# 🎯 Objetivo do Módulo

Capacitar o uso das principais estruturas e recursos da linguagem Python aplicados à manipulação e processamento de dados, permitindo:

- Construção de lógica computacional sólida
- Manipulação eficiente de estruturas de dados
- Desenvolvimento de algoritmos reutilizáveis
- Preparação para etapas mais avançadas como ETL, análise e machine learning

---

# 🗂️ Estrutura dos Módulos

O conteúdo está dividido em módulos progressivos, abordando desde operadores básicos até funções e organização de código:

---

## 📁 1. Operadores

Apresenta os operadores fundamentais da linguagem:

- Aritméticos
- Comparação
- Lógicos
- Atribuição
- Identidade
- Associação

📌 **Objetivo:** compreender como construir expressões e regras de decisão.

👉 Acesse: `./1-Operadores`

---

## 📁 2. Estruturas Condicionais e de Repetição

Aborda controle de fluxo e execução condicional:

- if, elif, else
- Estruturas aninhadas
- For e While
- Controle de loops (break e continue)
- Função range

📌 **Objetivo:** controlar o fluxo de execução e implementar lógica dinâmica.

👉 Acesse: `./2-IFs-For-While`

---

## 📁 3. Strings

Trabalha manipulação de dados textuais:

- Métodos de string
- Interpolação de variáveis
- Fatiamento
- Formatação
- Strings multilinha

📌 **Objetivo:** manipular e transformar dados textuais de forma eficiente.

👉 Acesse: `./3-Strings`

---

## 📁 4. Listas

Explora estruturas de dados mutáveis:

- Criação e acesso
- Fatiamento
- Iteração
- List Comprehension
- Métodos da classe list

📌 **Objetivo:** manipular coleções de dados de forma dinâmica.

👉 Acesse: `./4-Listas`

---

## 📁 5. Tuplas

Aborda estruturas imutáveis:

- Criação e acesso
- Fatiamento
- Iteração
- Diferenças entre listas e tuplas

📌 **Objetivo:** utilizar estruturas mais seguras e performáticas quando necessário.

👉 Acesse: `./5-Tuplas`

---

## 📁 6. Conjuntos (Sets)

Focado em operações com dados únicos:

- Eliminação de duplicidade
- Operações matemáticas (união, interseção, diferença)
- Métodos da classe set

📌 **Objetivo:** trabalhar com dados únicos e operações de conjunto.

👉 Acesse: `./6-Conjuntos`

---

## 📁 7. Dicionários

Estruturas chave-valor para dados estruturados:

- Criação e acesso
- Dicionários aninhados
- Iteração
- Métodos da classe dict

📌 **Objetivo:** representar dados estruturados e semi-estruturados.

👉 Acesse: `./7-Dicionario`

---

## 📁 8. Funções

Organização e reutilização de código:

- Definição de funções
- Parâmetros e retorno
- *args e **kwargs
- Escopo de variáveis
- Boas práticas

📌 **Objetivo:** modularizar código e construir soluções reutilizáveis.

👉 Acesse: `./8-Funcoes`
