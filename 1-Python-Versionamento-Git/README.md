# 📌 Módulos: Python e Versionamento com Git e GitHub

Este módulo reúne os fundamentos essenciais para o desenvolvimento em tecnologia, abordando desde a programação com Python até o controle de versão com Git e GitHub.

A proposta é fornecer uma base sólida tanto para escrita de código quanto para organização, versionamento e colaboração em projetos.

---

# 🗂️ Estrutura dos Módulos

O conteúdo está dividido em duas partes principais:

---

## 📁 1. Introdução ao Python

Aborda os conceitos fundamentais da linguagem Python, incluindo:

- Configuração do ambiente
- Tipos de dados
- Variáveis e constantes
- Entrada e saída de dados
- Conversão de tipos
- Primeiros programas

📌 **Objetivo:** construir a base para desenvolvimento e manipulação de dados.

👉 Acesse: `./1-Introducao-Python`

---

## 📁 2. Git e GitHub

Apresenta o funcionamento do versionamento de código na prática, incluindo:

- Conceitos de versionamento
- Uso do Git no dia a dia
- Principais comandos
- Trabalhando com branches
- Integração com GitHub
- Resolução de conflitos

📌 **Objetivo:** capacitar para controle de versão e trabalho colaborativo.

👉 Acesse: `./2-Git-Github`

---

# 🧪 Metodologia

Este módulo combina:

- Fundamentos teóricos
- Execução prática de comandos
- Exemplos reais de uso

O foco está em garantir entendimento aplicado, permitindo que os conceitos sejam utilizados em projetos reais.

Este módulo estabelece as bases essenciais para qualquer profissional de tecnologia, integrando programação e versionamento — dois pilares fundamentais para evolução em áreas como Engenharia de Dados, Desenvolvimento de Software e Machine Learning.
