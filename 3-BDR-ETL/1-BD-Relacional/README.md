# 🗂️ Módulo: Banco de Dados Relacional

Este diretório agrupa os fundamentos de bancos de dados relacionais, desde a modelagem e conceitos básicos até consultas avançadas em SQL.

---

# 🎯 Objetivo deste Agrupamento

- Compreender os conceitos fundamentais de bancos de dados relacionais e SGBDs
- Dominar a modelagem de dados com MER/DER e cardinalidades
- Aprender os comandos básicos do SQL (DDL, DML, DQL, DCL, DTL)
- Desenvolver consultas avançadas com JOINs, subconsultas e funções agregadas
- Aplicar normalização e índices para otimização de bancos de dados

---

# 📂 Estrutura Interna

| Pasta | Conteúdo | Status |
|-------|----------|--------|
| `./1-Bancos-Relacionais` | Introdução a bancos de dados, SGBDs, características ACID, modelagem MER/DER e configuração de ambiente no CloudClusters | ✅ Concluído |
| `./2-SQL-Basico` | Comandos fundamentais: CREATE TABLE, INSERT, SELECT, UPDATE, DELETE, PRIMARY KEY e FOREIGN KEY | ✅ Concluído |
| `./3-SQL-Avancado` | Normalização (1FN, 2FN, 3FN), JOINs (INNER, LEFT, RIGHT, FULL), subconsultas, funções agregadas e índices | ✅ Concluído |

---

# 🔗 Sequência Recomendada

Siga a ordem numérica dos diretórios:

1️⃣ **1-Bancos-Relacionais** → Conceitos teóricos e modelagem de dados
2️⃣ **2-SQL-Basico** → Operações CRUD e comandos essenciais
3️⃣ **3-SQL-Avancado** → Otimização, junções e consultas complexas

---

# 🧪 Projeto Integrador

Ao final deste agrupamento, você terá condições de:

- [x] Modelar um banco de dados relacional a partir de um problema real
- [x] Criar e estruturar tabelas com chaves primárias e estrangeiras
- [x] Realizar operações de inserção, consulta, atualização e exclusão
- [x] Executar consultas avançadas com JOINs entre múltiplas tabelas
- [x] Aplicar normalização para eliminar redundâncias
- [x] Otimizar consultas com índices
