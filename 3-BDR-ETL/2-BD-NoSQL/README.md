# 🗂️ Módulo: Banco de Dados NoSQL

Este diretório agrupa os fundamentos de bancos de dados NoSQL, com foco no MongoDB, desde a introdução aos conceitos até operações práticas de manipulação de dados.

---

# 🎯 Objetivo deste Agrupamento

- Compreender os conceitos de bancos de dados NoSQL e suas diferenças para bancos relacionais
- Conhecer os diferentes tipos de NoSQL (Key-Value, Document, Coluna, Grafo) e seus casos de uso
- Aprender a instalar e configurar o MongoDB Atlas na nuvem
- Dominar a modelagem de dados com documentos e coleções no MongoDB
- Aplicar operações de inserção, consulta, atualização e exclusão no MongoDB
