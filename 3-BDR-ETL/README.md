# 📌 Módulo: BDR e ETL

Este módulo consolida os fundamentos de bancos de dados relacionais e NoSQL, além dos processos de ETL (Extract, Transform, Load) para integração e preparação de dados.

O conteúdo foi estruturado para desenvolver a base necessária para atuação em áreas como:

- Engenharia de Dados
- Análise de Dados
- Business Intelligence
- Machine Learning

A abordagem prioriza entendimento prático, com foco em modelagem de dados, consultas SQL, operações NoSQL e construção de pipelines de dados.

---

# 🎯 Objetivo do Módulo

Capacitar o uso de bancos de dados e processos de integração aplicados à manipulação e preparação de dados, permitindo:

- Compreensão de bancos relacionais e NoSQL
- Modelagem de dados com MER/DER e cardinalidades
- Domínio de consultas SQL (básico e avançado)
- Operações práticas com MongoDB
- Construção de pipelines ETL com Python

---

# 🗂️ Estrutura dos Módulos

O conteúdo está dividido em módulos progressivos, abordando desde conceitos fundamentais de bancos de dados até pipelines ETL:

---

## 📁 1. Banco de Dados Relacional

Introdução aos conceitos fundamentais de bancos de dados relacionais, modelagem e SQL.

### Submódulos:

- **1-Bancos-Relacionais:** Conceitos de SGBD, características ACID, MER/DER, cardinalidades e configuração de ambiente no CloudClusters
- **2-SQL-Basico:** Comandos fundamentais (CREATE, INSERT, SELECT, UPDATE, DELETE) e chaves primárias/estrangeiras
- **3-SQL-Avancado:** Normalização (1FN, 2FN, 3FN), JOINs, subconsultas, funções agregadas e índices

📌 **Objetivo:** dominar modelagem e consultas em bancos relacionais.

👉 Acesse: `./1-BD-Relacional`

---

## 📁 2. Banco de Dados NoSQL

Introdução aos bancos de dados não relacionais com foco no MongoDB.

### Submódulos:

- **1-Introducao-MongoDB:** Conceitos NoSQL, tipos de bancos (Key-Value, Document, Coluna, Grafo), MongoDB Atlas e modelagem com Inner Documents e Referências
- **2-Operacoes-MongoDB:** Comandos práticos (insert, find, update, delete), operadores lógicos e de comparação, projeção, ordenação e paginação

📌 **Objetivo:** compreender e operar bancos NoSQL para cenários de alta escalabilidade.

👉 Acesse: `./2-BD-NoSQL`

---

## 📁 3. Fundamentos de ETL

Conceitos teóricos e práticos sobre processos de Extração, Transformação e Carga de dados.

### Conteúdos abordados:

- Introdução ao ETL e diferenças entre Data Mart, Data Warehouse, Data Lake e Lakehouse
- Etapas do ETL (Extração, Transformação, Carregamento)
- Vantagens e ferramentas de ETL
- Biblioteca Pandas para manipulação de dados
- Biblioteca Scikit-learn para modelos preditivos
- Framework Luigi para pipelines de dados em Python

📌 **Objetivo:** construir pipelines de dados eficientes e escaláveis.

👉 Acesse: `./3-Fundamentos-ETL`

---

## 📁 4. Pipeline ETL com Python e IA (Projeto Prático)

Aplicação prática de todos os conceitos aprendidos no módulo.

### Estrutura do projeto:

- Extração de dados de fontes diversas
- Transformação com Pandas e regras de negócio
- Carregamento em Data Warehouse
- Integração com IA para enriquecimento dos dados
- Pipeline automatizado com Luigi

📌 **Objetivo:** consolidar conhecimentos em um projeto completo de ponta a ponta.

👉 Acesse: `../../7-Projetos-Práticos/1-Pipeline-ETL-Python-IA`

---

# 🧪 Projeto Integrador

Ao final deste módulo, você terá desenvolvido um **Pipeline ETL completo com Python e IA**, aplicando:

- Extração de dados de fontes reais
- Transformação e limpeza de dados com Pandas
- Enriquecimento de dados com inteligência artificial
- Carregamento estruturado para análise
- Automação do pipeline com framework Luigi

Este projeto integra todos os conceitos do módulo em uma solução funcional e escalável.

---

# 📊 Boas Práticas Aplicadas

Durante este módulo, são reforçadas práticas essenciais do mercado:

- Modelagem de dados consistente (MER/DER)
- Normalização para evitar redundâncias
- Uso adequado de índices para otimização de consultas
- Escolha correta entre SQL e NoSQL conforme cenário
- Documentação de pipelines ETL
- Versionamento de código para processos de dados

# 🚀 Conclusão

Este módulo estabelece a base técnica necessária para trabalhar com bancos de dados e pipelines de dados no contexto de engenharia de dados e analytics.

A partir desses fundamentos, o profissional estará preparado para evoluir para:

- Análise de dados com Excel e ferramentas de BI
- Computação em nuvem com AWS

- Construção de soluções orientadas a dados em produção
