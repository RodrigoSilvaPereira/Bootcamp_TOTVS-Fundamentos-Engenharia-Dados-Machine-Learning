# 📌 Módulo: Computação em Nuvem

Este módulo consolida os fundamentos de computação em nuvem com foco na Amazon Web Services (AWS), abrangendo desde conceitos introdutórios até serviços essenciais de infraestrutura, computação, redes, armazenamento e bancos de dados.

O conteúdo foi estruturado para desenvolver a base necessária para atuação em áreas como:

- Engenharia de Nuvem (Cloud Engineering)
- Arquitetura de Soluções AWS
- DevOps
- Engenharia de Dados
- Infraestrutura como Código (IaC)

A abordagem prioriza entendimento prático, com foco em arquiteturas escaláveis, seguras e de alta disponibilidade.

---

# 🎯 Objetivo do Módulo

Capacitar o uso dos principais serviços de computação em nuvem da AWS, permitindo:

- Compreensão dos conceitos fundamentais de cloud computing
- Domínio da infraestrutura global AWS (Regiões, Zonas de Disponibilidade, Edge Locations)
- Utilização de serviços de computação como EC2, Auto Scaling e Load Balancing
- Implementação de redes virtuais com Amazon VPC
- Gerenciamento de armazenamento com EBS, S3 e EFS
- Operação de bancos de dados relacionais (RDS) e NoSQL (DynamoDB)
- Análise de Big Data com Amazon Redshift

---

# 🗂️ Estrutura dos Módulos

O conteúdo está dividido em módulos progressivos, abordando desde conceitos introdutórios de nuvem até serviços especializados da AWS:

---

## 📁 1. Cloud

Apresenta os conceitos fundamentais da computação em nuvem:

- Definição e modelo "as a service"
- Comparação entre On-Premise, Cloud e Híbrido
- Modelos IaaS, PaaS e SaaS
- Regiões e Zonas de Disponibilidade

📌 **Objetivo:** compreender os fundamentos da computação em nuvem.

👉 Acesse: `./1-Cloud`

---

## 📁 2. AWS Infraestrutura

Introdução à infraestrutura global da AWS:

- Regiões e Zonas de Disponibilidade
- Pontos de Presença (Edge Locations)
- Provisionamento de recursos (Console, CLI, SDKs)
- Elastic Beanstalk e CloudFormation (IaC)

📌 **Objetivo:** conhecer a infraestrutura global AWS e formas de provisionamento.

👉 Acesse: `./2-AWS-Infraestrutura`

---

## 📁 3. AWS Computação

Serviços de computação da AWS:

- Amazon EC2 e tipos de instância
- EC2 Auto Scaling (preditivo e dinâmico)
- Elastic Load Balancing (ALB, NLB, GLB, CLB)
- Serviços de mensageria (SQS, SNS, MQ, Kinesis, Pinpoint)
- Computação serverless (Lambda, Fargate, EventBridge)
- Containers (ECS e EKS)

📌 **Objetivo:** dominar os serviços de computação e escalabilidade na AWS.

👉 Acesse: `./3-AWS-Computacao`

---

## 📁 4. AWS Redes

Conceitos de redes na AWS:

- Amazon VPC (Virtual Private Cloud)
- Sub-redes públicas e privadas
- Gateways de Internet e Direct Connect
- Network ACLs (stateless)
- Security Groups (stateful)

📌 **Objetivo:** projetar redes isoladas e seguras na AWS.

👉 Acesse: `./4-AWS-Redes`

---

## 📁 5. AWS Armazenamento

Serviços de armazenamento e banco de dados da AWS:

- Tipos de armazenamento (objetos, arquivos, blocos)
- Amazon EBS (volumes persistentes)
- Amazon S3 e classes de armazenamento
- Amazon EFS (sistema de arquivos compartilhado)
- Amazon RDS (bancos relacionais gerenciados)
- Amazon DynamoDB (NoSQL serverless)
- Amazon Redshift (Big Data e Data Warehouse)

📌 **Objetivo:** gerenciar dados e armazenamento em diferentes cenários.

👉 Acesse: `./5-AWS-Armazenamento`
